# React Projekt - Kommandon

## Köra projektet
```
npm run dev
```
Startar localhost (vanligtvis port 5173)

## Andra kommandon
```
npm run build    - Bygger för produktion
npm run preview  - Testar production build
npm run lint     - Kollar kod-kvalitet
```

## Första gången
```
npm install      - Installerar allt som behövs
npm run dev      - Startar projektet
```

## Om något är fucked
```
Remove-Item -Recurse -Force node_modules, package-lock.json
npm install
```
