# boilerroom-project-3
Personalized Productivity System



# User Stories
#### Som användare vill jag...

## Övergripande
* Logga min produktivitet samt vad jag gjort
* Se historik för att motivera mig samt förstå mina mönster
* Vill redigera tidigare pass om jag skrev fel

## Tidsspårning
* Kunna starta samt avsluta timer som loggar tiden däremellan (start, paus, spara)
* Se historik över mina pass
* Automatiskt logga tid samt datum baserat på när jag använder timern.

## Energi och produktivitet
* logga min energinivå för att veta hur kämpigt det var
* Jag vill se när timern är på

## Fokuslägen
* sätta fokusläge som loggas i olika färger (deep work, möte, paus)

## Användarvänlighet
* Att den skall vara enkel att använda, inte för många funktioner 
* att appen/hemsidan skall vara tillgänglig på flera enheter
* kunna redigera tidigare pass om jag skrev fel
*  Växla mellan dark/light mode
